# Getting started
- Clone the repository
```
git clone https://github.com/KarmaChild/Community-Based-Organization-ReactJs-NodeJs-MySQL.git
```
- Install dependencies
```
cd backend
npm install
cd frontend
npm install

``` 

- Build and run the project
```
cd frontend
npm start
cd backend
node server.js
```
Navigate to `http://localhost:3000/`
